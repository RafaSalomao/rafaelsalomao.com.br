TECNOLOGIAS QUE UTILIZO

HTML, ou HyperText Markup Language, é uma linguagem de marcação utilizada para estruturar o conteúdo das páginas da web, definindo elementos como cabeçalhos, parágrafos, links e imagens.

CSS é uma linguagem de estilo utilizada para estilizar elementos HTML e controlar o layout e a apresentação das páginas da web.

JavaScript é uma linguagem de programação utilizada para tornar páginas da web interativas e dinâmicas, oferecendo funcionalidades como validação de formulários, animações e manipulação do DOM.

Python é uma linguagem de programação versátil, conhecida por sua sintaxe clara e legibilidade, amplamente utilizada em uma variedade de domínios, desde desenvolvimento web e científico até automação de tarefas e inteligência artificial..

C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft, comumente usada para desenvolvimento de software na plataforma .NET.

SQL é uma linguagem de consulta utilizada para interagir com bancos de dados relacionais, permitindo a criação, modificação e consulta de dados armazenados.

Git é um sistema de controle de versão distribuído, amplamente utilizado no desenvolvimento de software para rastrear e gerenciar alterações no código-fonte de projetos.
